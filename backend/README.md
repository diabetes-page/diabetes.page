## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the backend

```bash
# development mode
$ npm run start:dev

# watch mode
$ npm run start:dev:watch

# debug mode
$ npm run start:dev:debug

# production mode
$ npm run start:prod
```

## Test

```bash
# tests
$ npm run test
```
